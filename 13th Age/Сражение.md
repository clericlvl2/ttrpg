# Сражение

Сражения работают в пошаговом режиме - герои и их противники предпринимают действия по очереди. Как только очередь подошла к концу - всё начинается по новой, пока не будет определен исход битвы.

В начале боя все участники совершают бросок инициативы: `d20 + Инициатива`. Персонаж с самым высоким результатом ходит первый, с самым низким - последний.

## Что можно делать в свой ход?

Действия в бою делятся на ([несколько типов](#табличка-типы-действий)). В начале раунда игрок должен решить, какие способности и в какой комбинации он будет использовать.

**Бюджет хода:** 1 Standard Action (STD) + 1 Movement Action (MOV) + 1 Quick Action (QUICK)

- Действия можно использовать в любом порядке
- STD можно заменить на MOV или QUICK
- MOV можно заменить на QUICK

### Базовый список способностей

- **Умение персонажа (STD / MOV / QUICK).**
    - Атака, манёвр, заклинание, любая другая способность - смотри описание, чтобы узнать тип

- **Rally / Воодушевление (STD).**
    - 1/Battle
    - Лечение за 1 пункт Recovery
    - Скажи вслух, что заставляет тебя не сдаваться
    - Повторное ралли: потрать QUICK на проверку 11+
        - Успех: можешь сделать ралли
        - Провал: QUICK потрачен зря

- **Помощь (STD).**
    - Добавь +2 к проверке союзника - атаке, сейву или проверке навыка
    - Расскажи, как именно ты помогаешь

- **Перемещение (MOV).**
    - Переместиться неподалеку
    - Для Nearby - 1 MOV
    - Для Far Away - 2 MOV

- **Disengage / Отход (MOV).**
    - Безопасно выйти из ближнего боя
    - Проверка 11+, сложность +1 за каждого врага в Engage кроме первого
        - Успех: выходишь из Engage и перемещаешься на 1 MOV в рамках этого же действия
        - Провал: стоишь как стоял
    - **Pop Free:** некоторые способности позволяют выйти из Engage без проверки Disengage. Но ты всё ещё рядом с врагом - для того, чтобы стать Nearby, нужно использовать MOV

## Между ходами

- **Intercept / Перехват (INT).**
    - Перехватить противника, проходящего мимо тебя. Броска не нужно, вы теперь в Engage
    - Если для перехвата нужен MOV, он входит в действие

- **Opportunity Attack / Атака по возможности (FREE).**
    - Свободная атака по противнику, который пытается выйти из ближнего боя без Disengage
    - Базовая no-trigger melee attack, не активирует способности

## Особые действия

- **Fight in Spirit / Биться с чувством (SPEC).**
    - 1/Round
    - Если ты не участвуешь в бою физически - мёртв, отсутствуешь, не бросаешь кубики
    - Дай +2 к атаке или защите любому союзнику
    - Опиши, как вдохновляешь их мысленно
    - Повторно в том же бою: только +1

- **Retreat / Отступление (SPEC).**
    - Решение всей группы, можно объявить в любой момент
    - Все смертельные эффекты немедленно прекращаются
    - Полное излечение для каждого героя
    - Группа выходит из боя - опиши как
    - Группа терпит сюжетное поражение

## Кубик Эскалации (Escalation Die)

Специальный счётчик, значение которого прибавляется к проверкам атаки героев во время сражения. Кубик появляется на поле боя во 2-м раунде со значением 1. Каждый последующий раунд, значение увеличивается на 1, т.е. в 3-ем раунде 2, в 4-м раунде 3, и так далее до 6.

- Появляется во 2-м раунде боя со значением 1
- С каждым новым раундом к значению прибавляется +1
- Даёт бонус ко всем броскам атаки героев равный значению кубика
- Некоторые умения реагируют на чётные (2, 4, 6) или нечётные (1, 3, 5) значения
- Некоторые монстры тоже реагируют на его значение
- Если игроки бездействуют и выжидают - значение снижается или сбрасывается в 0

## Дистанции в бою

В бою есть три ([типа дистанций](#табличка-дистанции-в-бою)): Engaged (очень близко), Nearby (близко) и Far Away (далеко).

Если ты в ближнем бою - ты Engaged. Это ограничивает твои действия:

- Используешь дальнюю атаку - провоцируешь Opportunity Attack
- Используешь обычное заклинание - провоцируешь Opportunity Attack
- Пытаешься переместиться - провоцируешь Opportunity Attack (спасает Disengage)
- Атак и заклинания с дальностью **Close Quarters** используются без штрафов
- Нельзя сделать перехват (INT) между своими ходами

## Частотность способностей

В описании каждой активной способности героя есть указание, как часто её можно применять.

| Метка      | Как часто                                                   |
|------------|-------------------------------------------------------------|
| At-Will    | Каждый раунд в свой ход                                     |
| 2/Battle   | Два раза за бой                                             |
| 1/Arc      | Один раз за арку                                            |
| Recharge X | Один раз за бой, после боя бросок d20 - при X+ возвращается |

Для справки - единицы длительности:

- **Arc** - веха или арка в сюжете, обычно 3-4 битвы
- **Battle** - сражение, сопровождается броском инициативы
- **Round** - время, за которое все участники сражения успели что-то сделать
- **Turn** - ход конкретного участника сражения во время раунда

## Табличка. Дистанции в бою

| Зона                  | Описание                           | Перемещение |
|-----------------------|------------------------------------|-------------|
| **Engaged (В бою)**   | Ближний бой, вплотную к противнику | -           |
| **Nearby (Рядом)**    | На расстоянии одного перемещения   | 1 MOV       |
| **Far Away (Далеко)** | На расстоянии двух перемещений     | 2 MOV       |

## Табличка. Типы действий

| Тип              | Сокращение | Что это                                     | Можно заменить на |
|------------------|------------|---------------------------------------------|-------------------|
| Standard Action  | STD        | Атаки, заклинания, что-то серьезное         | MOV или QUICK     |
| Move Action      | MOV        | Переместиться на 1 зону дистанции           | QUICK             |
| Quick Action     | QUICK      | Мелочь: достать оружие, открыть дверь       | -                 |
| Interrupt Action | INT        | Действие между ходами, 1 раз за раунд       | -                 |
| Free Action      | FREE       | Сказать, прикинуть - быстро, в любой момент | -                 |
| Special          | SPEC       | Особая категория, отдельные правила         | -                 |
