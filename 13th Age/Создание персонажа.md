# Character Creation / Создание персонажа

Определи характеристики героя и постепенно заполни лист персонажа:

1. Определи свои Ability Scores:
   - Сила / Strength (STR)
   - Ловкость / Dexterity (DEX)
   - Телосложение / Constitution (CON)
   - Мудрость / Wisdom (WIS)
   - Интеллект / Intelligence (INT)
   - Харизма / Charisma (CHA)
2. Вычисли основные параметры персонажа: 
   - Armor Class / AC
   - Physical Defence / PD
   - Mental Defence / MD
   - Hit Points / HP
   - Initiative
   - и так далее (стр. 26 Heroes Handbook)
3. Открой [описание своего класса](./Classes.md), прочитай и запиши умения
    - Class Features
    - Talents
    - Powers / Spells (если есть)
4. Запиши снаряжение:
	- выбери оружие (подробнее в описании класса)
	- выбери доспех (подробнее в описании класса)
	- впиши себе 1 зелье лечения 
5. Открой описание своего происхождения (эльф, человек и тд), прочитай и выбери Kin Power
6. Выбери усиление (Adventurer Feat):
   - либо для 1-й способности от класса
   - либо для 1-й способности от происхождения
   - либо выбери Universal Feat со стр. 211 Heroes Handbook
7. Добавь итоги 0-й встречи и другие аспекты:
    - [One Unique Thing](./One%20Unique%20Thing.md)
    - [Backgrounds](./Backgrounds.md)
    - [Лики / Icons](./Icons.md)
    - Команда и связи
    - Имя персонажа

### Определение Ability Scores

Распредели 6 значений по характеристикам (STR, CON, DEX, INT, WIS, CHA), затем добавь **+2 к любым двум** на выбор.

**Выбери один из трёх способов:**

| Способ | Значения | Бонус                   |
|---|---|-------------------------|
| Стандартный набор | 17, 15, 14, 13, 12, 10 | +2 к любым двум хар-кам |
| Сбалансированный набор | 15, 15, 15, 14, 14, 12 | +2 к любым двум хар-кам |
| С кубиками | 18, 16 - фиксированные; остальные 4 - бросок 4d6, убрать наименьший | без бонуса              |

### Подробнее о классах

Класс - это архетип персонажа. В разделе с классом есть много умений, из которых персонаж собирается как конструктор. Что там будет:

**I. Class Features.**
Их может быть мало, может быть много, эти умения у вас просто есть, они составляют основу класса.

**II. Talents.**
Особые навыки, определяют стиль игры. Обычно нужно выбрать 3 штуки. Если не хочется выбирать, то есть набор по умолчанию.

**III. Powers / Spells.**
Крутые способности которые расширяют арсенал героя с "я бью" до "я жесть больно бью". Обычно 3 штуки, у каждого класса своё. Если не хочется выбирать, то есть набор по умолчанию.

**IV. Feats.**
Это разнообразные усиления талантов, заклинаний, рассовых и классовых бонусов. Обычно фиты перечислены под описанием способности. Фиты бывают разного уровня силы (Adventurer, Champion, Epic). На 1-м уровне ты можешь взять только 1 усиление с пометкой "Adventurer Feat" для какой-то 1-й способности. Также есть общие фиты без привязки к способностям и классам, можно заглянуть и туда (стр. 211 Heroes Handbook).

### Способности персонажей 

В описании каждой активной способности героя есть указание, как часто её можно применять. Ниже примеры и их объяснение:

- **At-Will** - каждый раунд в свой ход
- **2/Battle** - два раза за бой
- **1/Arc** - один раз за арку
- **Recharge X** - один раз за бой, после боя бросок d20 - при X+ способность возвращается

Какие в целом бывают длительности?

- **Arc** - веха или арка в сюжете, обычно 3-4 битвы
- **Battle** - сражение, сопровождается броском инициативы
- **Round** - время, за которое все участники сражение успели что-то сделать
- **Turn** - ход конкретного участника сражения во время раунда

> [Совет]
> У персонажей может быть много активных способностей (3–10). Полезно кратко описать каждую на отдельном листочке или стикере и разложить перед собой - как руку в карточной игре.
